css positioning

check out the blocks and the elements

position: static;

this is the default position, whereby the blocks appear in the order they are coded in the HTML file

check out the red and gray boxes above

position: fixed;

check out the orange block to the right - it stays fixed in the browser without moving as you scroll

we can move that block to any fixed position using right: left: top: etc

position: absolute;

and finally, the blue blocks (light and dark) are positioned on the page (not just as per the HTML default), but they move with the page
